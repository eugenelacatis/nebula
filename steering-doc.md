# NEBULA

## 1. Project Vision

Nebula turns any MP3 into a personalized journey through deep space. Drop a song, and a real-time audio analysis pipeline extracts bass, mids, highs, BPM, spectral energy, and everything possible while a background call to Claude reads the song's metadata and generates a scene seed — colors, warp speed, star density, cosmic tension etc. — that shapes the cosmos to match the music's cultural identity. The result is a Three.js warp tunnel where stars streak, nebulae pulse flares in sync with every beat. An audio waveform is shown, reacting to each beat of the song. The user is essentially flywing towards the wave and the wave is coming towards the user's screen. It twists and turns depending on the users dragging. If the user drags from center to right (turning left), it bends left (like a train track) and then continues straight.
